Start using the sandbox

Welcome to the Sandbox . To make it easier for you to get started, we’ve already provisioned a team with a sample app so that you can start using the platform right away.

Postman is an API platform for building and using APIs. You can download the Postman app here .

Download Postman collection

To help you make your first API calls, download the generic Postman collection containing sample requests to our APIs. You will also need the Postman environment file containing the credentials of your (demo) App.

Make your first API call

Everything should now be ready for you to make your first API call. Simply import the files into Postman, make sure you select the Getting started Environment, and follow the steps of the collection.

You are ready to go!

Congratulations! You can now start experimenting with the sandbox, access the API catalogue to check out all the available APIs.
