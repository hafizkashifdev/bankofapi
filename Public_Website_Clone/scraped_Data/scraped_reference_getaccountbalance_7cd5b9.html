Get account balance

get https://api.truelayer.com/data/v1 /accounts/ /balance

Get balance for specified account

Path Params
string
required

unique ID for an account or card

Query Params
string

When set to true, data is fetched asynchronously. Expect 202 status code

uri

When async is set to true, a webhook will be POSTed to the URI provided

Headers
string

Client-specified ID used to debug requests

string

IP address of the end user requesting data. Necessary to overcome 4 calls per day rate limits

Responses

Response body
object
results
array of objects
results
object
string
required

Currency used for this account or transaction

EUR GBP USD AUD

number

The amount of funds available to the account holder. This takes into account any pending transactions and includes any overdraft facility.

number
required

The amount of funds within the account. This does not include any pending transactions.

number

Pre-arranged overdraft limit. Available for all “TRANSACTION” and “BUSINESS_TRANSACTION” account types.

string

Last update time of the data

string

Indicates whether the request was successful. 'Queued', 'Running' and 'Failed' can be returned for async requests

Succeeded Queued Running Failed
