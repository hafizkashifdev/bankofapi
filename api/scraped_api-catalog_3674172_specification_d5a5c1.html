API contract definition for Customer Discovery. The goal of these endpoints is to find potential matches for a particular customer entity given certain search criteria.

Servers

Individual

Discover if an individual is a retail banking customer of NatWest.

POST ​/individual
Retrieve a "Match Pending" or "No Match" status for a requested customer.

Returns a "Match Pending" status if at least one customer is found with an exact match against the requested given_name, family_name and birthdate. Otherwise a "No Match" status is returned.

No parameters

Details of the individual you are attempting to discover.
