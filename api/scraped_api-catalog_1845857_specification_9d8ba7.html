Indicative Quotes

GET ​/quote​/indicative
Get Indicative Quote

Parameters

Name Description
authorisation *
string
( header )

Authorization Token

fromCurrency *
string
( query )

ISO code of the currency you want to convert from.

Available values : AED, AUD, BBD, BGN, BHD, BMD, BRL, CAD, CHF, CNY, CZK, DKK, EGP, EUR, GBP, HKD, HUF, ILS, INR, ISK, JOD, JPY, KES, KRW, KWD, LKR, MAD, MUR, MXN, MYR, NOK, NZD, OMR, PHP, PKR, PLN, QAR, RON, SAR, SEK, SGD, THB, TND, TRY, TWD, USD, ZAR

Example : GBP

toCurrency *
string
( query )

ISO code of the currency you want to convert to.

Available values : AED, AUD, BBD, BGN, BHD, BMD, BRL, CAD, CHF, CNY, CZK, DKK, EGP, EUR, GBP, HKD, HUF, ILS, INR, ISK, JOD, JPY, KES, KRW, KWD, LKR, MAD, MUR, MXN, MYR, NOK, NZD, OMR, PHP, PKR, PLN, QAR, RON, SAR, SEK, SGD, THB, TND, TRY, TWD, USD, ZAR

Example : EUR

amount
number
( query )

This parameter can be used to specify the amount you want to convert. If an amount is not specified, then value of 1 is assumed.

Example : 1

inverse
string
( query )

If ‘true’ then the API will also include the inverse rate. An inverse rate is a quote for which the ‘from’ currency and ‘to’ currency are switched. An inverse is calculated by dividing one by the exchange rate. Example: If the exchange rate for $1 USD to GBP = 0.70586041, then the inverse rate would be 1/0.70586041 = 1.4167107

Available values : false, true

Default value : false

Example : true

amountCurrency
string
( query )

ISO code of the currency for the amount provided in Amount parameter. MANDATORY - if the amount is provided in the Amount parameter. Available values: Either the value provided in fromCurrency or in the toCurrency parameter.

Example : GBP

decimalPlaces
number
( query )

This parameter can be used to specify the number of decimal places included in the output. Example 1 USD to GBP = 0.70586041 with decimalPlaces=3, the output returned will be EUR = 0.706

Example : 3

margin
string
( query )

This parameter can be used to add a margin (-/+) to the spot rate provided by the system. E.g. add margin=2.05 parameter to the endpoint and the API will return our spot rate plus the margin of 2.05 percent.

Example : 1.05
